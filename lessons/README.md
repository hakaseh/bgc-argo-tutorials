# Lessons [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/hakaseh/bgc-argo-tutorials/HEAD)

This folder contains a series of lessons written in Python and Jupyter Notebook, which allows you to use coding to learn about BGC-Argo data analysis. Click on the **Binder** icon above to try it out! If you find it useful, you may install it on your computer (with `git clone` or direct download) so that you can customize and save your work. Also, please feel free to provide your user experience by visiting [this link](https://forms.gle/oAGmz5RTW4Pp46bt7) at any time.

Below is the list of lessons currently available:

* Lesson 1: Search for BGC-Argo float profiles based on our research interests 🔍
* Lesson 2: Draw maps and timelines to understand spatial and temporal coverages 🎨
* Lesson 3: Download float time series and individual profiles ⬇️
* Lesson 4: Take a first look at the float time series and QC flags 👀
* Lesson 5: Smoothing and NPQ correction of CHLA
* Lesson 6: Decomposition of BBP700
* Lesson 7: Interpolate the float time series 🪄
